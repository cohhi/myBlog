javaWeb项目搭建

汉化eclipse 添加动态项目 安装并配置tomcat 随便写个东西,项目跑起来~~~

汉化eclipse

首先打开你的 eclipse 点击help-install New Software
点击add添加url
https://download.eclipse.org/technology/babel/update-site/latest/
选择chinese(simple),点击finish
安装完成后重启eclipse,然后就是中文了

添加动态项目

首先点击 帮助-安装新软件 work with填写
2022-12 - https://download.eclipse.org/releases/2022-12/
勾选Web,XML,java EE adn OSGi ENterprise Development
下载安装后重启eclipse
选择新建-动态web项目(没有的话去其它项里面找)

安装并配置tomcat

首先下载并解压tomcat10 tomcat https://tomcat.apache.org/download-10.cgi 打开eclipse
选择首选项
选择 服务器-运行时环境-点击添加
选择-Apache-Apache Tomcat v10.0-勾选创建一个本地服务器-点击完成-应用并关闭

随便写个东西,项目跑起来~~~

鼠标右键点击项目
点击创建jsp
向body中写入一点东西
右击运行到服务器
等待服务器启动......
启动成功后浏览器输入localhost:8080/项目名/xxx.jsp
